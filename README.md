**Good-Wolf Search** is a Google-inspired search engine interface designed with a modern **Glassmorphism UI** and dynamic aesthetics. Built to showcase proficiency in **DOM Manipulation**, **CSS Animations**, and **API Integration** without relying on heavy frameworks.

### üé® Visuals & UI

### ‚ö° Functionality
* **üéô Smart Voice Search:**
    * Powered by the **Web Speech API**.
    * **Context-Aware:** Syncs with the selected language (e.g., if you select "Japanese", the voice recognition switches to Japanese).
    * **Error Handling:** Includes a built-in debugger that alerts users if the microphone is blocked or disconnected.
* **üåè Language Filtering:** Forces Google Search results to prioritize the selected language.

## üì• Installation (For Developers)

If you want to run the code locally on your machine:

1.  **Clone this repository**
    Open your terminal and run:
    ```bash
    git clone [https://github.com/CooodeddbyPJ/Goodwolf.git](https://github.com/CooodeddbyPJ/Goodwolf.git)
    ```

2.  **Navigate to the project folder**
    Enter the folder you just downloaded:
    ```bash
    cd Goodwolf
    ```

3.  **Run the project**
    Simply open the `index.html` file in your browser (Google Chrome or Edge is recommended for Voice Search compatibility).

## üöÄ Usage

1.  **Type** your query in the glowing search bar.
2.  **Select a Language** (Optional) to filter results (e.g., select "Filipino" for local results).
3.  Click **Search** to view Google results.
4.  Click **I'm Lucky** to go straight to the first website.
5.  Click the **Microphone Icon** to speak your search.
