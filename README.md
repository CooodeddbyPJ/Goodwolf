## üìñ Overview
**Good-Wolf Search** is a reimagined search interface designed with a modern **Glassmorphism UI**. It transcends the traditional search bar by integrating **Robust Voice Search**, **Language-Specific Querying**, and dynamic aesthetics without relying on heavy frameworks.

It serves as a technical showcase for **DOM Manipulation**, **CSS Animations**, and **Web Speech API** integration.

## ‚ú® Key Features

### üé® Visuals & UI
* **Glassmorphism Design:** A modern, translucent interface featuring blur effects and crisp white borders.
* **Dynamic Background:** Smooth, animated gradients that breathe life into the page.
* **Responsive Layout:** Adaptive Flexbox design that centers perfectly on both Desktop and Mobile.
* **Interactive Motion:** Custom CSS entrance animations and hover states.

### ‚ö° Functionality
* **üéô Smart Voice Search:**
    * Powered by the native **Web Speech API**.
    * **Context-Aware:** Automatically syncs with your selected language (e.g., selecting "Japanese" switches voice recognition to Japanese).
    * **Debug-Ready:** Built-in alerts if the microphone is blocked or disconnected.
* **üåè Language Filtering:** Forces Google Search results to prioritize the region/language you select.
* **üçÄ "I'm Lucky" Button:** Direct navigation that skips the search results page.
* **‚ú® Smart UX:** "Clear Text" button dynamically appears only when typing is detected.

## üì• Installation (Local Development)

To run this project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/CooodeddbyPJ/Goodwolf.git](https://github.com/CooodeddbyPJ/Goodwolf.git)
    ```

2.  **Navigate to the directory:**
    ```bash
    cd Goodwolf
    ```

3.  **Launch:**
    Open `index.html` in your browser.
    > *Note: Chrome or Edge is recommended for full Voice Search compatibility.*

## üöÄ How to Use

1.  **Type** your query in the glowing search bar.
2.  **(Optional)** Select a **Language** to filter results (e.g., "Filipino" for local results).
3.  Click **Search** to view Google results.
4.  Click **I'm Lucky** to jump straight to the top result.
5.  Click the **Microphone Icon** to use voice commands.
